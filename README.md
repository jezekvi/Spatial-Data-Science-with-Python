# Spatial Data Science with Python

## Cíle kurzu
Cílem kurzu je, aby byl student schopen zpracovat prostorová data v prostředí Jupyter Notebook, porozuměl systému správy verzí GIT a uměl využít volně dostupné kódy z platformy GitHub pro vlastní práci s prostorovými daty.

## Harmonogram semestru

| Týden | Téma | Klíčové kompetence |
| :--- | :--- | :--- |
| **01** | **Úvod do prostředí** | Konfigurace VS Code, představení cloudových řešení, Markdown dokumentace, Jupyter Notebook. |
| **02** | **Git a GitHub** | Verzování kódu, správa repozitářů, kolaborativní vývoj. |
| **03** | **GIS: Geopandas** | Vektorové datové struktury, prostorové dotazy a operace. |
| **04** | **DPZ: Základní aplikace a spektrální analýza** | Práce s rastry, spektrální příznaky, výpočet vegetačních indexů. |
| **05** | **DPZ: API a Cloud-Native analýza** | Přístup k datům přes API, architektura STAC, cloudové zpracování. |
| **06** | **DPZ: Klasifikace pomocí ML** | Aplikace algoritmů strojového učení. |
| **07–12** | **Semestrální práce** | Obhajoba projektů. |
